Features

Everything you need, nothing you don’t

○

Menu-bar only

Memo lives in your menu bar. No Dock icon, no window clutter — always available, never in the way.

△

Global hotkey

Press and hold ⌥ Space from any app to start recording. Release to transcribe. Fully customisable in Settings.

◆

Whisper-quality accuracy

Powered by OpenAI’s gpt-4o-mini-transcribe model. Fast, accurate, and multilingual across 18+ languages.

■

Edit before pasting

A floating panel lets you review and edit the transcription before it’s pasted. Hit ⌘⏎ to paste or Esc to cancel.

●

Auto-paste mode

Enable auto-paste in Settings to skip the review panel entirely and paste instantly after transcription.

△

Secure API key storage

Your OpenAI API key is stored in the macOS Keychain — never in plain text, never on disk unprotected.

How it works

Three steps, start to paste

1

Download & install

Download the DMG, drag Memo into Applications, and open it. A microphone icon appears in your menu bar.

2

Add your OpenAI API key

Open Memo Settings from the menu bar and paste your OpenAI API key. It’s saved to the Keychain immediately.

3

Hold ⌥ Space and speak

Hold the hotkey anywhere on your Mac, dictate, then release. Memo transcribes and pastes the text into the active field.
